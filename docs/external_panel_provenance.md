# German integrated external panel

The file `data/external/germany_real_covid_validation_input.csv` was imported from the existing
Bayesian SVEIAHR project. Its embedded `source` field identifies the construction as RKI cases/deaths,
RKI hospitalization data, DIVI ICU occupancy, and OWID vaccination data. The admissions-based
hospitalization series is a proxy, not direct ward occupancy. The integrated panel is generated by
`scripts/integrate_external_panel.py`; it is not a fresh download and should retain the original
source citations and release dates in any publication.
